# Установка библиотек для обработки PDF и PPTX

Перед использованием функции загрузки материалов курса необходимо установить необходимые библиотеки.

## Установка

Выполните в терминале в корне проекта:

```bash
npm install pdfjs-dist jszip
npm install --save-dev @types/jszip
```

## Использование

1. Откройте раздел "Академия" → "КУРСЫ"
2. Нажмите кнопку "Загрузить материалы курса (PDF, PPTX)"
3. Выберите все ваши PDF и PPTX файлы (можно выбрать несколько)
4. Нажмите "Обработать и интегрировать материалы"
5. Материалы будут автоматически интегрированы в курс с контрольным листом

## Формат файлов

- **PDF файлы**: Поддерживаются все стандартные PDF
- **PPTX файлы**: Поддерживаются презентации PowerPoint (формат .pptx)

## Рекомендации по именованию файлов

Для правильной сортировки материалов рекомендуется именовать файлы с числовым префиксом:
- `01_Введение.pdf`
- `02_Состояние_1.pptx`
- `03_Состояние_2.pdf`
- и т.д.

Файлы будут отсортированы по числовому префиксу и интегрированы в курс в правильном порядке.
