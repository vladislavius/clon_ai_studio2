# Материалы курса "Состояния"

Эта папка предназначена для хранения материалов курса по состояниям.

## Как добавить файлы:

### Вариант 1: Скопировать файлы в эту папку
1. Скопируйте все 100 файлов в папку `data/course_materials/states/`
2. Убедитесь, что файлы имеют понятные имена (например: `01_введение.txt`, `02_состояние_1.txt` и т.д.)

### Вариант 2: Указать путь к существующей папке
Если файлы уже находятся в другом месте на вашем компьютере, укажите полный путь к папке.

### Вариант 3: Загрузить через интерфейс
Файлы можно будет загрузить через интерфейс Академии (функция будет добавлена).

## Формат файлов:
- Поддерживаются форматы: `.txt`, `.md`, `.docx`, `.doc`
- Рекомендуется использовать `.txt` или `.md` для лучшей обработки

## Структура:
После добавления файлов они будут автоматически интегрированы в интерактивный курс с контрольным листом.
