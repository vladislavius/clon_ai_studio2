# Руководство по загрузке материалов курса

## Шаг 1: Установка библиотек

Выполните в терминале:

```bash
npm install
```

Это установит все необходимые библиотеки, включая `pdfjs-dist` и `jszip` для обработки PDF и PPTX файлов.

## Шаг 2: Загрузка материалов

1. Откройте приложение
2. Перейдите в раздел **"Академия"** → **"КУРСЫ"**
3. Нажмите кнопку **"Загрузить материалы курса (PDF, PPTX)"**
4. Выберите все ваши PDF и PPTX файлы (можно выбрать несколько файлов сразу)
5. Нажмите **"Обработать и интегрировать материалы"**

## Шаг 3: Использование курса

После обработки материалов:
- Весь материал будет объединен в один курс "Курс по состояниям"
- Автоматически создастся контрольный лист с заданиями
- Каждое задание будет связано с соответствующим материалом
- Студенты смогут изучать материал и отмечать выполненные задания

## Формат файлов

- **PDF**: Поддерживаются все стандартные PDF файлы
- **PPTX**: Поддерживаются презентации PowerPoint (формат .pptx)

## Рекомендации по именованию

Для правильной сортировки материалов рекомендуется именовать файлы с числовым префиксом:

- `01_Введение.pdf`
- `02_Состояние_1.pptx`
- `03_Состояние_2.pdf`
- `04_Практика.pptx`
- и т.д.

Файлы будут отсортированы по числовому префиксу и интегрированы в курс в правильном порядке.

## Функции курса

- ✅ Интерактивный контрольный лист с заданиями
- ✅ Отслеживание прогресса выполнения
- ✅ Чат с куратором (заглушка)
- ✅ Видеосвязь с куратором (заглушка)
- ✅ Возможность отправки ответов на задания
- ✅ Проверка заданий куратором (заглушка)

## Изображения и графики

Система автоматически извлекает и отображает:
- ✅ **Из PDF**: Все страницы рендерятся как изображения (сохраняются все графики, диаграммы, схемы)
- ✅ **Из PPTX**: Все изображения извлекаются из презентации
- ✅ Изображения можно увеличить по клику
- ✅ Автоматическая оптимизация размера для быстрой загрузки

## Примечания

- Обработка больших файлов может занять некоторое время
- Рекомендуется загружать файлы по частям, если их очень много (100+)
- Текст из PDF и PPTX извлекается автоматически
- Форматирование может быть частично потеряно при извлечении текста
- Изображения сохраняются в формате base64 и встраиваются в курс
- Для PDF каждая страница рендерится как изображение, что сохраняет все графики и диаграммы
